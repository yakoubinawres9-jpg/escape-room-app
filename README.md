Solve clever AI-powered riddles, choose your theme, test your wits under time pressure, and escape before your lives run out!

## Features
- Multiple themes: Classic, Science, Pirate
- Difficulty levels: Easy, Medium, Hard
- Timer & lives system
- Hints for each riddle
- AI-powered answer checking with fallback to local pool

## How to Run
1. Clone the repo
2. Install backend dependencies: `npm install` in `escape-room-backend`
3. Install frontend dependencies: `npm install` in `escape-room-frontend`
4. Start backend: `npm run start`
5. Start frontend: `npm run dev`
6. Open `http://localhost:5173` in your browser
